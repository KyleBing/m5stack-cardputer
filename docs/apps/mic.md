# Mic 麦克风

主菜单按键：`r`

实时示波器波形 + VU 电平 + 增益调节。v1.0.0 起**不再**支持 TF 录音。

![mic-scope](/shots/mic-scope.png)

## 快捷键

| 按键 | 作用 |
|------|------|
| `-` `=` 或 `,` `.` | 增益 − / + |
| `h` | Help（进 Help 会关麦；关闭 Help 后恢复） |

## 使用说明

1. 进入即开麦画波形；注意环境噪声与增益，避免削波。
2. 退出时会静默回收 Speaker / I2S，减轻嗡嗡声。
3. 录音相关功能已移除；只需看电平与波形时用本 App。
