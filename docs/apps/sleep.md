# Sleep 睡眠

主菜单按键：`s`

进入浅睡（Light）或深睡（Deep）前的倒计时确认页。

## 快捷键

| 状态 | 按键 | 作用 |
|------|------|------|
| 浅睡倒计时 | **BtnGO** | 立即唤醒 / 取消进入（以界面为准），将不会释放内存 |
| 浅睡倒计时 | `s` | 改为深睡 |
| 深睡倒计时 | **BtnGO** | 唤醒说明：深睡唤醒会重启 |
| 任意倒计时中 | `ESC` / `GO` | 取消，回主菜单 |

## 使用说明

1. 进入后先进入 **Light** 倒计时；期间可按 `s` 切到 **Deep**。
2. **浅睡**：可较快唤醒，适合短时省电。
3. **深睡**：更省电，唤醒后设备会重启回到主流程。
4. 倒计时结束前按 `ESC` 可放弃睡眠。
