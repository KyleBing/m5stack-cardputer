# 功能目录

按主菜单分类浏览。每个 App 文档均包含：**简介与截图预留**、**快捷键**、**使用说明**。

截图命名：`docs/public/shots/{app}-{子功能}.png`。

## 智能家居

| 键 | App | 说明 |
|----|-----|------|
| `m` | [Mijia](./mijia) | 米家本地 / BLE 设备控制，列表·宫格·编组·快捷键 |
| `x` | [Infrared](./infrared) | TV / AC 红外遥控（GPIO44） |

## 网络与配置

| 键 | App | 说明 |
|----|-----|------|
| `u` | [Config](./config) | SoftAP / LAN Web 配网与设备管理 |
| `w` | [WiFi](./wifi) | 连接、扫描、切换已保存档案（最多 5 条） |
| `b` | [BLE](./ble) | 附近 BLE 设备扫描列表 |

## 时间与电源

| `s` | [Sleep](./sleep) | 浅睡 / 深睡倒计时 |
