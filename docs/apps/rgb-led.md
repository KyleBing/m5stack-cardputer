# RGB LED

主菜单按键：`l`

测试板载 RGB LED：开关、颜色、亮度。与 LCD 背光电源相关，离开 App 会关灯并恢复背光逻辑。

## 快捷键

| 按键 | 作用 |
|------|------|
| `t` | 开关切换 |
| `0` | 关 |
| `1` | 红（亦见 tip 的 R） |
| `2` | 绿 |
| `3` | 蓝 |
| `1`–`7` | 预设颜色（Help） |
| `-` `=` | 亮度 − / + |
| `h` | Help |

## 使用说明

1. 进 App 后用数字键选颜色，`-`/`=` 调亮度，`t` 开关。
2. 注意与屏幕共用电源路径时的亮度变化属预期行为。
3. `ESC` 退出并清理 LED 状态。
