# Battery 电池

主菜单按键：`p`

显示实时电量，并绘制约 **1 小时**内的电量历史曲线（含睡眠段标记）。

## 截图

![battery-main](/shots/battery-main.png)

## 快捷键

本 App 底栏主要为图例说明，无专用字母操作键。

| 图例 | 含义 |
|------|------|
| 1h log | 历史采样 |
| live | 实时段 |
| sleep | 睡眠期间 |
| now | 当前点 |

返回：`ESC` / `GO`。翻页键无额外页面时无效。

## 使用说明

1. 进入即可查看当前电量百分比与曲线。
2. 历史在后台采样；深睡等场景会在曲线上区分。
3. 用于观察续航与睡眠耗电趋势，不作精确电量计校准工具。
