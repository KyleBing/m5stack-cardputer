## Shortcuts

| Key | Action |
|-----|--------|
| `s` | Start / rescan |
| Arrow keys · `;,.` `/` | Page |
| `[` `]` | Page (noted in Help; may be omitted from main tip) |
| `h` | Help (device type notes) |

## Usage

1. Press `s` to scan; results are listed across pages.
2. The list uses a small font; index and category use different colors.
3. Help covers common types: generic devices / beacon / ble-svc, etc.
4. For Mijia BLE sensors, press `r` on the specific device inside [Mijia](./mijia). This App is a general nearby-device browser.
