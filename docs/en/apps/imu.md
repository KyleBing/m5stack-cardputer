# IMU

Main menu key: `g`

BMI270 accelerometer / attitude visualization (XY / Z traces).

## Shortcuts

No dedicated letter shortcuts; tilt the device to move the points. Back: `ESC` / `GO`.

## Usage

1. Opens with a live motion trace.
2. Use to confirm the IMU works or demo attitude sensing.
3. Leaving the App clears drawing state.
