## Shortcuts

| Key | Action |
|-----|--------|
| Arrow keys · `;,.` `/` | Previous / next pattern |

The footer shows the current pattern name and page hint.

## Usage

Use to check dead pixels, refresh, and color. Page through patterns; there are no other settings.
