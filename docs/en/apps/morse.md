# Morse

Main menu key: `j`

Press letter keys to play the matching Morse code tone; pitch is adjustable.

## Shortcuts

| Key | Action |
|-----|--------|
| `a`–`z` and other letters | Play that letter’s Morse |
| `-` | Lower frequency |
| `=` | Raise frequency |

## Usage

1. Press letters to sound them; volume follows global `sound.volume`.
2. Use `-` / `=` to a comfortable pitch.
3. Handy for practice or a simple beacon demo.
