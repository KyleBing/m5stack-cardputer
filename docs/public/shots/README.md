# 截图目录（shots）

将设备实拍图放到本目录，文件名与文档中的 `/shots/...` 引用一致。

当前仓库内已有同名 **1×1 占位 PNG**（保证 VitePress 能构建）；用 `Fn+s` 实拍图覆盖即可，无需改文档路径。

## 命名规则

```
{app}-{子功能}.png
```

示例：`mijia-detail.png`、`time-clock.png`、`infrared-ac.png`。
