# 截图目录（shots）

将设备实拍图放到本目录，文件名与文档中的 `/shots/...` 引用一致。

可用固件内 `Fn+s` 截屏，经 Config `/shots` 下载后按下列命名放入本目录。

## 命名规则

```
app_{app}_{子功能}.png
```

示例：`app_mijia_list.png`、`app_cursor_24h.png`、`app_ir_ac.png`。
